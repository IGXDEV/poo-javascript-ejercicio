# Sistema de Figuras Geométricas en JavaScript (POO)

Este proyecto demuestra conceptos avanzados de Programación Orientada a Objetos.

## Características
- **Herencia Multinivel:** `Figura2D` y `Figura3D`.
- **Patrones de Diseño:**
  - Singleton (Gestor de Figuras)
  - Builder (Construcción de objetos complejos)
- **Validaciones:** Encapsulamiento y protección de datos.
- **Polimorfismo:** Renderizado ASCII y cálculos de área.

## Ejecución
`node sistema-figuras.js`
